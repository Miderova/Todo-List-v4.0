# To-Do App "Акулы"

Это стильный и современный To-Do список с прогрессбаром и конфетти, выполненный на HTML, CSS и JavaScript. Приложение сохраняет задачи в localStorage и позволяет удобно управлять списком дел.

## Особенности

- Добавление новых задач.
- Редактирование существующих задач.
- Удаление задач.
- Отмечание задач как выполненные.
- Прогрессбар с процентом выполненных задач.
- Индикатор выполнения в формате `Выполнено / Всего`.
- Анимация конфетти при завершении всех задач.
- Сохранение задач в `localStorage` — задачи не теряются после перезагрузки страницы.
- Адаптивный и современный интерфейс с красивым фоном, прозрачными блоками и мягкими тенями.

## Технологии

- HTML5
- CSS3 (Flexbox, прозрачности, градиенты)
- JavaScript ES6 (модули)
- LocalStorage
- [tsparticles.confetti](https://www.npmjs.com/package/tsparticles-confetti) — для конфетти
